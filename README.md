**# 🚀 DevOps CI/CD Pipeline Project**



**## 📌 Project Overview**

**This project demonstrates a basic \*\*DevOps workflow\*\* by containerizing a Python Flask application using Docker and managing the source code with Git and GitHub.**



**The goal of this project is to simulate a \*\*real-world DevOps process\*\* including version control, containerization, and application deployment.**

**## ▶️ How to Run the Project**



**### 1️⃣ Clone Repository**



**git clone https://github.com/YOUR-USERNAME/Devops-my-app.git**



**### 2️⃣ Navigate to Project Folder**



**cd Devops-my-app**



**### 3️⃣ Build Docker Image**



**docker build -t devops-my-app .**



**### 4️⃣ Run Docker Container**



**docker run -p 5000:5000 devops-my-app**
